A 3D airplane visualization project built with **React Three Fiber** and **Three.js**. This project demonstrates 3D model rendering, animations, and interactive controls in a web environment.

## Demo

You can add a live demo link here if hosted (e.g., GitHub Pages, Netlify).

## Features

- Load and render a 3D airplane model (`.glb` or `.gltf`) using `@react-three/drei`.
- Basic airplane movement and rotation controls.
- Smooth animation and interactive camera controls.
- Responsive design for desktop and mobile browsers.

## Technologies Used

- **React.js**
- **React Three Fiber**
- **Three.js**
- **@react-three/drei**
- **Tailwind CSS / Bootstrap** (if used)
- **GLTF/GLB 3D model files**
